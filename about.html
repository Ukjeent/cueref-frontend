About CueRef

CueRef is a metadata extraction and matching tool for music used in video production. It helps editors, producers, and rights managers extract track information from EDL files and match it with metadata from various music libraries like Epidemic Sound, APM Music, and Extreme Music.

This project is built to streamline the workflow of verifying cue sheet data, making music clearance and reporting easier and more accurate.

This is an early version focused on EDL uploads and metadata preview. More features are coming soon.
